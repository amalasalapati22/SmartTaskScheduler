# Smart Task Scheduler with Priority Queues

A simple JavaFX desktop application to manage tasks based on urgency. Users can add, view, and delete tasks using a GUI. Internally, it uses Javaâ€™s `PriorityQueue` to sort tasks by priority.

## Features

- Add new tasks with title and priority (1 = High)
- View all tasks in priority order
- Delete the top-priority task
- JavaFX GUI-based, no console needed

##  Tools Used

- Java 17+
- JavaFX SDK 21
- Eclipse IDE
- GitHub
